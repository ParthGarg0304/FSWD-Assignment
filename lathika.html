HTML - Hyper Text Markup Language

HTML 5 is the latest version of Hyper Text Markup Language, the standard language used to create and design webpages.
It introduces new elements and attributes that provide more flexibility and functionality for web developers.
Some of the key features of HTML5 include improved support for multimedia (such as audio and video elements), new form controls, semantic elements
